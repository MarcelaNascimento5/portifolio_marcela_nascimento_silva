# 🖋️ PRISMA - Editor Markdown Minimalista
 
## 📝 Descrição do Projeto
O **PRISMA** é um editor de Markdown projetado para ser "jovem e divertido", quebrando a monotonia dos editores técnicos tradicionais. Ele oferece uma experiência de escrita fluida com visualização em tempo real, permitindo que criadores de conteúdo e desenvolvedores foquem na estrutura do texto enquanto acompanham o resultado final instantaneamente.
 
Desenvolvido com uma estética vibrante (Branco, Verde Limão, Rosa e Azul), o sistema integra ferramentas de manipulação de arquivos, histórico de alterações (undo/redo) e uma interface dinâmica que une funcionalidade e design moderno.

## 🚀 Tecnologias Utilizadas
* **Linguagens:** TypeScript, HTML5, CSS3
* **Bibliotecas:** React 19, Tailwind CSS 4, Framer Motion (animações), React Markdown, Lucide React (ícones)
* **Ferramentas:** Vite, Firebase (Autenticação e Sincronização), npm
 
## 📊 Resultados e Aprendizados
O projeto alcançou uma excelente performance de renderização e uma usabilidade intuitiva, focada na experiência do usuário.
* **Acurácia de Visualização:** O preview em tempo real reflete 100% da sintaxe Markdown suportada de forma instantânea.
* **Design System Customizado:** Utilizei o Tailwind CSS 4 para criar um tema personalizado e responsivo com uma estética "Y2K/Bento Grid" moderna.
* **Arquitetura de Estado:** Implementação de um sistema robusto de histórico de ações que garante que nenhuma alteração seja perdida.
 
 
## 🔧 Como Executar
1. Clone o repositório.
2. Instale as dependências: `npm install`.
3. Execute o comando: `npm run dev`.
